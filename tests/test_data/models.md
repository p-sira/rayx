# Model Information

## Perfect Suzanne

Perfect Suzanne is the default monkey mesh in Blender that we have cleaned up the open edges to ensure manifold.

- File: `perfect-suzanne.stl`
- Triangles: 698

## Bunny (Low, Medium, High)

The Bunny is from [The Stanford 3D Scanning Repository](https://graphics.stanford.edu/data/3Dscanrep/).

- File: `bunny_zipper_res4.stl` (Low)
- Triangles: 908

- File: `bunny_zipper_res3.stl` (Medium)
- Triangles: 3,851

- File: `bunny_zipper_res2.stl` (High)
- Triangles: 16,214

## Thai Statue

The Thai Statue is from [The Stanford 3D Scanning Repository](https://graphics.stanford.edu/data/3Dscanrep/). We have applied a decimation of 0.01 to the mesh to reduce the triangle count.

- File: `thai-statue-decim001.stl`
- Triangles: 100,000
